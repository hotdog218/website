NOTICE: Steam-based servers currently rely on UDP ports. UDP ports do not reply when information is sent to them. Because of this, any game server listed below that runs via steamCMD, will NOT have an accurate online/offline status displayed. A solution is currently in the works, but for now you'll have to just ask me or test the connection for yourself.

Server Statuses are updated every 15 minutes.
